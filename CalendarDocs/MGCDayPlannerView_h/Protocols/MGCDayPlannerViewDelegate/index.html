Introduction

The MGCDayPlannerViewDelegate protocol defines methods that allow you to manage the selection of events in a day planner view and respond to operations like scrolling and changes in the display. The methods of this protocol are all optional.


Methods

-dayPlannerView:didDeselectEventOfType:atIndex:date:

Tells the delegate that the specified event was deselected.

-dayPlannerView:didEndDisplayingDate:

Tells the delegate that the specified day is not displayed anymore in the day planner view.

-dayPlannerView:didEndScrolling:

Tells the delegate when a scrolling operation on the day planner view concludes.

-dayPlannerView:didScroll:

Tells the delegate that the day planner view was scrolled.

-dayPlannerView:didSelectEventOfType:atIndex:date:

Tells the delegate that the specified event was selected.

-dayPlannerView:shouldSelectEventOfType:atIndex:date:

Asks the delegate if the specified event should be selected.

-dayPlannerView:willDisplayDate:

Tells the delegate that the specified day is about to be displayed in the day planner view.

-dayPlannerViewDidZoom:

Tells the delegate that the day planner view was zoomed in or out, increasing or decreasing the hour slot height.

dayPlannerView:didDeselectEventOfType:atIndex:date:

Tells the delegate that the specified event was deselected. 

@optional

- (void)dayPlannerView:(MGCDayPlannerView*)view 
        didDeselectEventOfType:(MGCEventType)type atIndex:(NSUInteger)index 
        date:(NSDate*)date;
Parameters
view

The day planner view object notifying about the selection change.

type

The type of the event.

index

The index of the event.

date

The starting day of the event.

Return Value

YES if the event should be selected or NO if it should not.

Discussion

The day planner view calls this method when the user successfully deselects an event. It does not call this method when you programmatically deselect the event.

dayPlannerView:didEndScrolling:

Tells the delegate when a scrolling operation on the day planner view concludes. 

@optional

- (void)dayPlannerView:(MGCDayPlannerView*)view 
        didEndScrolling:(MGCDayPlannerScrollType)scrollType;
Parameters
view

The day planner view object in which the scrolling occured.

scrollType

Indicates what was scrolled through (days, time or both).

Discussion

The method is only called when the view is scrolled horizontally. It is called after user interaction and when the view is scrolled programmatically.
